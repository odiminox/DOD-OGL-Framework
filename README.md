# DOD-OGL-Framework
Data Orientated Design OpenGL C++ Game Framework I created in 2013

Features
- Bullet Physics engine integration
- DOD design for entity and object processing
- 3D rendering
- 2D rendering
- Custom cameras
- Basic Blinn-Phong reflection model
- Multiple lights
- Questionable C++ code
